Descrição detalhada

Definições e funções relacionadas à CPU virtual do sistema HeliOS.

Este arquivo contém as estruturas e funções necessárias para gerenciar a CPU virtual, incluindo execução de instruções, manipulação de registradores e operações na pilha.

Documentação dos tipos

◆ CpuInfo

typedef struct CpuInfo CpuInfo

Estrutura que contém informações resumidas sobre a CPU.

Esta estrutura é usada ao destacar um processo da CPU.

enum CpuState

Valores de enumerações

CPU_IDLE 

A CPU está ociosa.

CPU_HALTED 

A CPU foi interrompida.
