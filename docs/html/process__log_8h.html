Descrição detalhada

Funções para log de informações sobre a execução de processos.

Este arquivo contém as definições das funções responsáveis por registrar informações sobre a execução dos processos. O objetivo é permitir o acompanhamento do comportamento dos processos, como o escalonamento e a execução de algoritmos.

Função para escrever logs de execução de processos.

Esta função é responsável por escrever logs contendo informações sobre o comportamento dos processos, como os tempos de execução, a ordem de execução e os algoritmos utilizados. Esses logs são úteis para análise posterior da performance do sistema.

Parâmetros
algorithNameNome do algoritmo de escalonamento utilizado.
fileNameNome do arquivo onde o log será escrito.
processArray de estruturas ProcessStatistics contendo informações sobre os processos.
logCountNúmero de logs que serão gravados no arquivo.
executionOrderString que define a ordem de execução dos processos (ex: "FIFO", "SJF").
