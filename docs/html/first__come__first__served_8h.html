Descrição detalhada

Implementação do gerenciador de processos utilizando o algoritmo First-Come, First-Served (FCFS).

Este arquivo define as estruturas e funções necessárias para gerenciar processos utilizando o algoritmo FCFS, no qual os processos são atendidos na ordem de chegada.
