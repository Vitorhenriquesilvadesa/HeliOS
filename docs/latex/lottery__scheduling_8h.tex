\doxysubsection{Descrição detalhada}
Implementação do gerenciador de processos utilizando o algoritmo Lottery Scheduling. 

Este arquivo define as estruturas e funções necessárias para gerenciar processos utilizando o algoritmo Lottery Scheduling, onde cada processo recebe uma quantidade de tickets e o processo a ser executado é escolhido de forma probabilística com base nos tickets.
