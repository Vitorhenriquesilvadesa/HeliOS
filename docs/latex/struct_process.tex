\doxysubsection{Descrição detalhada}
Estrutura que representa um processo no sistema. 

A estrutura \doxylink{struct_process}{Process} contém todos os dados necessários para a execução de um processo no sistema, incluindo seu identificador, estado, tempos de execução e as informações sobre o programa a ser executado.
