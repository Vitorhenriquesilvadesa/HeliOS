\doxysubsection{Descrição detalhada}
Definições e interfaces principais para o sistema operacional virtual Heli\+OS. 

Este arquivo contém as estruturas, tipos e funções essenciais para a criação e gerenciamento do sistema operacional virtual Heli\+OS. Ele fornece suporte para gerenciamento de processos, criação de sistemas e execução de programas associados.
