\doxysubsection{Descrição detalhada}
Estrutura que representa uma lista de processos. 

A estrutura \doxylink{struct_process_array}{Process\+Array} é usada para armazenar múltiplos processos no sistema, permitindo operações de gerenciamento e manipulação de processos, como inserção e remoção.
