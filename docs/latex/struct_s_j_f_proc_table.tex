\doxysubsection{Descrição detalhada}
Estrutura que mantém a tabela de processos para o escalonamento Shortest Job First. 

A tabela contém uma lista de processos prontos para execução, ordenada de acordo com o tempo de execução estimado dos processos. O processo com o menor tempo de execução é o primeiro na fila.
