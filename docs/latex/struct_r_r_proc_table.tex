\doxysubsection{Descrição detalhada}
Estrutura que mantém a tabela de processos para o escalonamento Round Robin. 

A tabela contém uma lista de processos prontos para execução. Também mantém o controle do processo atualmente em execução (head, tail, current) e o tamanho da fila.
