\doxysubsection{Descrição detalhada}
Estrutura que representa um nó da lista encadeada de processos no escalonamento Round Robin. 

Cada nó na lista representa um processo que está esperando para ser executado. Ele contém o PID do processo e um ponteiro para o próximo nó na lista.
